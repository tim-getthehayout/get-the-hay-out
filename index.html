Application summary
Cull animal
Animal will be removed from its group and marked inactive. All history (weight, calving, lineage) is preserved. You can reactivate from the animal record if needed.
Reset all data
This will permanently erase everything:
· All animals, groups, and classes
· All pasture events and locations
· All feed types, batches, and inventory
· All manure batches and input applications
· All tasks, users, and feedback
· All settings and operation name
If Drive is connected, the remote file will also be overwritten with empty data on next sync.
Consider exporting a backup first.
